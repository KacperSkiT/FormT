## Opis

Aplikację należy uruchomić przez docker-compose używając komendy 'docker-compose up -d'.

Udało sie zrealizować cały punkt 2 i punkt 3 dotyczący PHP (wysyłanie formularza na odwpowiendi adres e-mail), dodatkowo formularz zapisywany jest w bazie danych, w której są 2 tabele połączone relacją OneToMany: transports, items.
